===============================

starter react template project with react-router, browserHistory, es6 babel webpack and livereload

```sh
npm i
```

build

```sh
npm run serve
open http://localhost:3000
```

development

```sh
npm start
npm run dev
```

force stop

```sh
npm stop # after control+c
```
